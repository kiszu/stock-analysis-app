## 🎨 颜色说明

```
信号类型颜色:
💚 STRONG_BUY  → 绿色 (买入)
💚 BUY          → 绿色 (买入)
🧡 HOLD         → 橙色 (持有)
❤️ SELL          → 红色 (卖出)
❤️ STRONG_SELL   → 红色 (强烈卖出)

涨跌颜色:
+ 正数  → 绿色
- 负数  → 红色
```

## 📱 交互说明

| 交互 | 动作 |
|------|------|
| 点击股票卡片 | 打开详情页 |
| 点击搜索图标 | 打开搜索页 |
| 点击设置图标 | 打开设置页 |
| 点击返回按钮 | 返回上一页 |
| 下拉刷新 | 刷新数据 |
| 上拉加载 | 加载更多 |
